so a neural network is a system that consists of layers with each layer containing multiple neurons.
each layers output is the next layers input. the neurons in these layers just take the output of the previous layer and multiplies it by a weight and then compose the output using an activation function.

an easier way to understand neural network is to visualize it.

![](/3.%20deep_learning/assets/iclh-diagram-batch-01-03-deepneuralnetwork.png)

#### neural network structure 
- input layer --> the layer that inputs the data into the system.
- hidden layers --> the intermediate layers situated between the input and output layers of a           neural network. They are the "brains" of the network, responsible for extracting features,             learning complex patterns, and transforming raw data into representations that the final layer    can use to make predictions.
- output layer --> responsible for the final computations to produce the final prediction.
- connections --> they take the output of the previous layer and multiply it by weights (w) and         adds a bias (b).
- nodes --> they take the weighted input and applies an activation function to it 

#### the why?

in a neural network we stack couple of layers of multiple models on top of each other. but what difference does it make?? why not just use a single machine learning model and call it a day.

to understand why we stack models we first need to know the **hypothesis space**:

##### hypothesis space

in a simple linear regression problem we use a linear equation as a hypothesis function:

$$f(x) = y = xw + b $$

a hypothesis space is a set every single linear equation (hypothesis function) our model is capable of choosing to describe the data.

for example:
- $f(x) = 5x+ 10$
- $f(x) = 3x+ 5$
- $f(x) = 2x+ 7$
all of these functions exist in a space called the **hypothesis space**.

now back to why we stack the models together. 

if we use a single neuron as solution to the problem, the model will take the raw data as it is. as random ,messy and non-linear the data is, the model wont be able to bend the linear hypothesis space around these complex data boundaries to create the best fit function for the data leading to suboptimal predictions.

instead neural networks take the output of one single model and uses it as an input feature. 

$$\text{Raw Input } X \longrightarrow \text{Layer 1} (z^{[1]}) \longrightarrow \underbrace{\text{Activation } \sigma(z^{[1]})}_{\text{Space Warp}} \longrightarrow \text{Layer 2 } (z^{[2]})$$

By passing the data through an activation function at the end of each layer, we perform **geometric origami** on the data space. We shift, rotate, stretch, and bend the coordinate system.

so in other terms --> we use this chaotic, raw data space and alter it using the activation function by rotating, shifting or twisting it turning it from a random space into linear simple space we can use to later as an input for the output layer to make more accurate predictions.

#### single neuron model

before we can wrap our heads around a complex topic like neural networks that can have an enormous number of nodes and many many connections between them. it would help us to zoom in on a single neuron model.

##### individual neuron:

![](/3.%20deep_learning/assets/images(1).png)

so basically a neuron takes in the inputs features $x$ and multiplies them by a weight $\theta$ then adds a bias and then sums them up.

$$f(x) = z =b+ \sum_{i=1}^n X.\theta$$

up until here this matches the normal hypothesis formula for a simple machine learning model.

but as we discussed before we will need to add an activation function to alter the hypothesis space so we will compose the f(x) inside another activation function as follows.

$$\underbrace{A}_{\text activation} = \sigma(z) = \sigma(b+ \sum_{i=1}^n X.\theta)$$

the activation function used here $\sigma$ is the **sigmoid function**.

and the loss function becomes:

$$L(\theta) = \sum_{i=1}^n (y-A)^2$$
