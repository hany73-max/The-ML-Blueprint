# Forward Pass

The forward pass is the process where input data flows through the neural network starting from the input layer, passing through hidden layers where non-linear activation functions are applied, and finally reaching the output layer to produce a prediction ($y$).

Before diving deep into the forward pass, let's establish the notation system:

![](/3.%20deep_learning/assets/ChatGPT_Image_Jul_6,_2026,_04_00_19_PM.png)

| **Component**      | **Notation** | **Meaning**                                                                      |
| ------------------ | ------------ | -------------------------------------------------------------------------------- |
| **Weight**         | $W^L_{jk}$   | Weight matrix for layer $L$, connecting source node $k$ to destination node $j$. |
| **Bias**           | $b^L_m$      | Bias vector for node $m$ in layer $L$.                                           |
| **Pre-activation** | $z^L_m$      | Linear combination ($W \cdot a + b$) for node $m$ in layer $L$.                  |
| **Activation**     | $a^L_m$      | Non-linear output ($\sigma(z)$) for node $m$ in layer $L$.                       |

### Step-by-Step Mathematical Example

![](/3.%20deep_learning/assets/ChatGPTImageJul6,2026,05_48_16PM.png)

We will look at a 4-layer neural network to understand how individual scalar numbers are calculated under the hood.

this tutorial involves the **vectorization**.

we do not feed samples into a network one by one. We stack multiple samples together into a **Batch**. This shifts our calculations from lone scalar arrays into an optimized, multi-dimensional matrix multiplication engine. the process of vectorizing is basically dealing with the data not as a scalars or single samples but instead as vectors and matrices which reduces the need for loops and simplifies the code by a lot.

The first layer consists of raw input features from $X_1$ to $X_n$. Passing these inputs into the **first node** of the first hidden layer yields the pre-activation score 
$Z_1^1$:

$$Z_1^1 = X_1 \cdot W^1_{11} + X_2 \cdot W^1_{12} + \dots + X_n \cdot W^1_{1n} + b^1_1 = b^1_1 + \sum_{i=1}^n X_i W^1_{1i}$$

Vectorized Framework:

$$Z^1 = \underbrace{X}_{m \times n} \cdot \underbrace{W^1}_{n \times h} + b^1$$ 

Where: 
- $m \rightarrow$ is the number of training samples (batch size) 
-  $n \rightarrow$ is the number of incoming input features 
-  $h \rightarrow$ is the number of processing neurons (nodes) in this hidden layer 

The weight matrix grid is structurally arranged so that features map down the rows, and individual destination neurons map across the columns: 

$$W^1 = \begin{bmatrix} W_{11} & W_{12} & \dots & W_{1h} \\ W_{21} & W_{22} & \dots & W_{2h} \\ \vdots & \vdots & \ddots & \vdots \\ W_{n1} & W_{n2} & \dots & W_{nh} \end{bmatrix} \quad \text{Shape: } (n \times h)$$ 

$$b^1 = \begin{bmatrix} b_1^1 & b_2^1 & \dots & b_h^1 \end{bmatrix} \quad \text{Shape: } (1 \times h)$$

The next step is to apply the activation function to add non-linearity and bend the space:

$$A_1^1 = \sigma(Z_1^1) = g\left(b^1_1 + \sum_{i=1}^n X_iW^1_{1i}\right)$$

This handles a single node. The complete output of the first hidden layer is an activation vector consisting of the outputs from all nodes in this layer:

$$A^1 = \begin{bmatrix} A_1^1 \\ A_2^1 \\ \vdots \\ A_n^1 \end{bmatrix}$$

vectorized:

$$A^1 = g(Z^1)$$

This vector $A^1$ now acts as the direct input features for the next layer (the second hidden layer), replacing the raw inputs $X$.

$$Z_1^2 = A_1^1 \cdot W^2_{11} + A_2^1 \cdot W^2_{12} + \dots + A_n^1 \cdot W^2_{1n} + b^2_1 = b^2_1 + \sum_{i=1}^n A^1_i W^2_{1i}$$

$$Z^2=A^1 \centerdot W^2+b$$

Applying the activation function to this layer's pre-activation gives:

$$A_1^2 = \sigma(Z_1^2) = g\left(b_1^2 + \sum_{i=1}^n {A_i^1 W_{1i}^2}\right)$$

We collect these to form the second hidden layer vector:

$$A^2 = \begin{bmatrix} A_1^2 \\ A_2^2 \\ \vdots \\ A_n^2 \end{bmatrix}$$

$$A^2= g(Z^2)$$

We continue this exact pipeline all the way to the output layer. The final prediction $y_1$ for the first output neuron is calculated as:

$$y_1 = \sigma(Z^3_1) = \sigma\left(b^3_1 + \sum_{i=1}^n{A^2_iW^3_{1i}}\right)$$

Leaving you with the final system prediction vector:

$$y = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}$$ 

$$Z^3=A^2 \centerdot W^3+b$$

$$y=\sigma(Z^3)$$

#### Scaling the Math: Activation Functions

Before translating these equations into Python, we must address how this math scales practically. First, the abstract activation function symbol ($\sigma$) must be defined based on its role in the network:

##### **ReLU** (Rectified Linear Unit):

Applied in hidden layers to introduce non-linearity by passing positive values and zeroing out negative signals ($g(z) = \max(0, z)$).

it is simply a piece-wise function:

$$g(z) = \begin{cases} z & \text{if } z > 0 \\ 0 & \text{if } z \le 0 \end{cases}$$

##### **Sigmoid:** 

Applied at the final output layer to squash continuous linear scores into strict probabilities between $0$ and $1$ ($\sigma(z) = 1 / (1 + e^{-z})$). explained in detail in logistic regression.

more on the activation functions later.

### putting it into coding

the nodes are nothing but weight multiplied by the input features and added a weight. so we can manage the nodes in each layer by  just deciding on the weights and bias matrices dimensions.

##### Weights

$$W = \begin{bmatrix} W_{11} & W_{12} & \dots & W_{1h} \\ W_{21} & W_{22} & \dots & W_{2h} \\ \vdots & \vdots & \ddots & \vdots \\ W_{n1} & W_{n2} & \dots & W_{nh} \end{bmatrix} \quad \text{Shape: } (n \times h)$$

Where $h$ is the number of neurons in the current layer and $n$ is the number of incoming input features from the previous layer. If we want to create a hidden layer with 100 nodes, we set $h = 100$, while $n$ dynamically adapts to the structural width of the data entering it.

```python 
import numpy as np 

# n = number of incoming input features 

# h = number of processing nodes inside this layer 

# The weights are assigned randomly. 
# The 0.01 multiplier scales them down to avoid exploding activations. 
W = np.random.randn(n, h) * 0.01
```
##### baises

the bias is quite similar 

$$b^1 = \begin{bmatrix} b_1^1 & b_2^1 & \dots & b_h^1 \end{bmatrix} \quad \text{Shape: } (1 \times h)$$

```python
import numpy as np

b = np.zeros((1, h))

```

##### the forward pass

$$Z=A \centerdot W+b$$

$$y=\sigma(Z)$$

```python
import numpy as np

current_input = X

for layer in layers:
	Z = np.dot(current_input, W) + layer.b
	
	# deciding on the activation function
	if layer.activation_function == "RELU":
		layer.A = np.maximum(0, Z)
	elif layer.activation_function == "sigmoid":
		layer.A = sigmoid(Z)
	
	# updating the input to be able to feed it to the next layer
	current_input = layer.A
```
